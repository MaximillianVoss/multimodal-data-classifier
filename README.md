# Структура папки ВКР

- `project/` - рабочий Python-проект приложения, который можно открывать в PyCharm.
- `docs/` - пояснительная записка, презентация, шаблоны и методические материалы.
- `.venv/` - локальное виртуальное окружение для запуска и проверки проекта.

Для разработки и запуска приложения открывайте папку `project`.
