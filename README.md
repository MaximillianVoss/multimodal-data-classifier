# Структура репозитория ВКР

- `project/` - рабочий Python-проект приложения для классификации и пакетной сортировки документов.
- `docs/` - пояснительная записка, презентация, шаблоны и сопроводительные материалы.
- `.venv/` - локальное виртуальное окружение для запуска и проверки проекта.

Для разработки и запуска приложения открывайте папку `project`.

## Быстрые команды из корня репозитория

```powershell
python main.py
python generate_assets.py
python scripts/build_thesis.py
python scripts/build_presentation.py
```
