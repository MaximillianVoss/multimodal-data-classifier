# Система интеллектуальной классификации и пакетной сортировки документов

Проект подготовлен для ВКР по теме разработки прикладного сервиса, который определяет тип документа по тексту, по скану и умеет сортировать целый ZIP-архив документов по категориям.

## Стек

- `FastAPI` для API и серверной части
- `Gradio` для low-code web-интерфейса
- `scikit-learn` для обучения текстовой и визуальной моделей
- `SQLite` для хранения истории запросов, пакетных прогонов и метаданных моделей
- `python-docx` для извлечения текста из DOCX при пакетной обработке
- `pytest` для автоматических тестов

## Структура

- `vkr_classifier` - исходный код приложения
- `tests` - автотесты
- `artifacts` - обученные модели, демонстрационные архивы, таблицы и графики
- `scripts` - генерация диаграмм, записки и презентации
- `../docs` - записка ВКР, презентация и методические материалы

## Быстрый запуск

```powershell
python -m pip install -r requirements.txt
python main.py
```

После запуска приложение будет доступно по адресу `http://127.0.0.1:8000`. Если порт `8000` уже занят, приложение автоматически выберет следующий свободный локальный порт.

## Поддерживаемые сценарии

- классификация текста документа;
- классификация скана документа;
- пакетная сортировка ZIP-архива по типам `Договор`, `Счет`, `Приказ`, `Служебная записка`, `Отчет`.

## Запуск тестов

```powershell
python -m pytest --cov=vkr_classifier --cov-report=term-missing
```
